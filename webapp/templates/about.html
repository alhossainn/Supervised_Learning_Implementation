About NeuroScan AI

Advanced Brain Cancer Classification using Deep Learning

Dataset Information

The model was trained on a comprehensive dataset of brain MRI scans categorized into three classes: Brain Glioma, Brain Meningioma, and Pituitary Tumor.
The dataset is available at: Kaggle Multi-Cancer Dataset

Model Architecture

This application uses a ResNet-50 convolutional neural network, a deep learning model with 50 layers, specifically adapted for medical image classification.

Architecture: ResNet-50

Interpretability with Grad-CAM

Grad-CAM (Gradient-weighted Class Activation Mapping) provides visual explanations for the model's predictions by highlighting the regions in the image that were most influential in the classification decision.

This helps medical professionals understand and trust the AI's reasoning process.
